# Unsharp_Masking
实验操作及处理步骤如下：<br/>
(1) 读入一幅原始灰度图像。<br/>
(2) 利用如式(1)或(2)所示的平滑滤波器，进行平滑滤波。<br/>

| |1|4|7|4|1|
|---|---|---|---|---|---|
||4|16|26|16|4|
|H2=1/273|7|26|47|26|7|
||4|16|26|16|4|
||1|4|7|4|1|

(3) 用原始图减去第(2)步处理结果，得到 Unsharp 掩模图像。<br/>
(4) 再用原始图加上 Unsharp 掩模图像（即上一步的结果），即得到反锐化处理图像。<br/>
